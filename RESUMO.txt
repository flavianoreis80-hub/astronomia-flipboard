🚀 INÍCIO RÁPIDO (3 PASSOS)
════════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAR
    $ pip install feedparser requests agendador --break-system-packages

2️⃣  TESTAR
    $ python3 main.py
    
3️⃣  AGENDAR (escolha uma opção)
    
    Linux/Mac:
    $ crontab -e
    # Adicione: 0 8 * * * cd ~/astronomia-flipboard && python3 main.py >> agendador.log 2>&1
    
    Windows:
    → Abra "Agendador de Tarefas"
    → Criar Tarefa (Diariamente às 08:00)
    
    Python:
    $ python3 agendador.py
